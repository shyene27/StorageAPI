Read Me

This is a Storage project with managing the data through API

Function #1

Cheking the Web Service availability
http://localhost:8080/StorageAPI/

Function #2

Doing a search by product_qty and total_price in orders table using @GET
http://localhost:8080/StorageAPI/api/v1/inventory/{prod_qty}/{total_price}

Function #3

Adding a new entry in orders table using @Post
http://localhost:8080/StorageAPI/post.html

Function #4

Updating a total_price field in a chosen entry of the order table using @Put
http://localhost:8080/StorageAPI/put.html

Function #5

Delete a chosen entry of the order table using @Delete
When deleting, the system also makes a copy of the deleted entry into the archiveorders table
http://localhost:8080/StorageAPI/delete.html

Function #6

Export
http://localhost:8080/StorageAPI/export_csv.jsp
